## Overview

AltTab Professional Suite is a complete window management and navigation platform that gives users full control over how they switch between open applications and windows, with the customization depth and open-source transparency that power users demand.

Multiple keyboard shortcut configurations allow different switching behaviors to be mapped to different key combinations, making it possible to have separate shortcuts for switching windows within an application versus across all applications.

Space and display filtering shows windows from specific Spaces or only from the current display, helping users in complex multi-Space or multi-monitor setups navigate quickly within the context they are working in.

Active community development means AltTab is continuously improved with new features, better compatibility with macOS updates, and responses to community-reported issues, keeping it reliable through every macOS version.

AltTab Professional Suite is the window navigation tool for Mac users who treat precise, fast window switching as a fundamental productivity capability and want the most capable, customizable switcher available on the platform.

## Performance & Integration

AltTab is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
